GIT远程仓库（U盘）

发表于 2018-01-09 | 更新于 2018-10-16 | 分类于 辅助类学习

总结一次在U盘上搭建Git的远程仓库过程。

  • Git初始化
  • 在project文件夹下创建项目等相关文件后
  • 通过mount命令获取U盘挂载路径：/Volumes/HyperX
  • 在U盘上创建Git仓库
  • 回到本地项目根目录下
  • 把当前项目推送至远程仓库
  • 如何pull down？
  • 新建本地repo
  • git init
  • git remote add “远程仓库名称” /Volumes/HyperX/GitRepo/Project
  • git pull “远程仓库名称” master
  • 注: 使用sourcetree可直接根据仓库路径拉取代码。
